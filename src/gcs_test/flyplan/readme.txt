共有四种类型的航点轨迹，经纬度单位为度（不使用度分秒），距离单位为米，偏航角单位为角度（北偏西为+0~+180度，北偏东-0～~180度），类型依次为1/2/3/4：
1.基于全局gps坐标，格式为（经度 纬度 相对原点高度 偏航角度 类型）；
2.基于全局距离坐标：以home点为原点，向右为x轴正方向，向前为y轴正方向，格式为（x距离 y距离 相对原点高度 偏航角度 类型）；
3.基于局部距离坐标：以整体航点任务开始时的无人机坐标为原点，向右为x轴正方向，向前为y轴正方向（不考虑当时无人机的偏航角），格式为（x距离 y距离 相对高度 偏航角度 类型）；
4.基于航点距离坐标：以每个航点任务开始时的无人机坐标为原点，向右为x轴正方向，向前为y轴正方向（不考虑当时无人机的偏航角），格式为（x距离 y距离 相对高度 偏航角度 类型）。
